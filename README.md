# Blacktop Bandits Website

## Setup & Deployment

### Local Development
```bash
npm install
npm start
```
Open http://localhost:3000

### Deploy to Netlify
1. Push this folder to a GitHub repo
2. Go to https://netlify.com → "Add new site" → "Import from GitHub"
3. Select your repo
4. Build settings: leave blank (static site)
5. Publish directory: `.`
6. Click **Deploy site**

### Images
Place your photos (BTB2.jpg, BTB3.jpg, BTB4.jpg, BTB5.jpg) in the same folder as index.html.
